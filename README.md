# Gakó Barello y Madame Véga (React)

## Correr en local

```powershell
cd C:\Users\javie\gakobarello
npm install
npm run dev
```

Abrí la URL que te da Vite (normalmente `http://localhost:5173`).

## Build de producción

```powershell
npm run build
npm run preview
```

## Dónde editar contenido

- Historia: `content.js` (raíz del proyecto)
- Media: colocá archivos en `public/media/` y agregalos en `media` dentro de `content.js`
